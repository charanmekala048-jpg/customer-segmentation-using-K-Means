# Customer Segmentation using K-Means Clustering

## Objective
The objective of this project is to segment customers into different groups based on their characteristics such as annual income and spending score. This helps businesses understand customer behavior and improve marketing strategies.

## Algorithm Used
K-Means Clustering is an unsupervised machine learning algorithm that groups similar data points into clusters.

## Steps Involved
1. Import the dataset
2. Select required features
3. Normalize the data
4. Use the Elbow Method to find the optimal number of clusters
5. Train the K-Means model
6. Assign cluster labels
7. Evaluate clustering performance
8. Visualize the clusters

## Evaluation Metrics
- **Silhouette Score**: Higher value indicates better clustering
- **Davies-Bouldin Index**: Lower value indicates better clustering
- **Calinski-Harabasz Index**: Higher value indicates better-defined clusters
- **Inertia**: Lower value indicates compact clusters

## Technologies Used
- Python
- Pandas
- Matplotlib
- Scikit-learn

## Conclusion
K-Means clustering is useful for customer segmentation because it groups customers with similar behavior. These segments can help businesses in targeted marketing and decision-making.
